## 3) Presentation rapide

Review Flow est une application SaaS de gestion d'avis clients.
Le projet permet de centraliser les contacts, gerer les campagnes et suivre les avis.

Stack principale:
- Next.js 16 (App Router)
- React 19
- Supabase (Auth + Postgres)
- Stripe (abonnement Pro)
- Resend (emails)


## 4) Installation locale

Installation:
```bash
npm install
```

## 5) Lancer le projet


```bash
npm run dev
```

Ouvrir:
- `http://localhost:3000`

Build production local:
```bash
npm run build
npm run start
```
